# AWS Demo Deployment

This deployment is for public/demo documents only. Indexing and retrieval run locally on the EC2 instance, but the final answer is generated by Groq, so retrieved document excerpts are sent to Groq.

## Instance

Use a small Ubuntu EC2 instance for a low-cost demo. AWS Free Tier credits or usage limits may apply to eligible accounts, but EC2 remains pay-as-you-go outside those credits and allowances.

## Setup

```bash
sudo apt update
sudo apt install -y python3.13 python3.13-venv nginx git
sudo mkdir -p /opt/insight-ai/data /opt/insight-ai/app
sudo chown -R ubuntu:ubuntu /opt/insight-ai

git clone https://github.com/danielcha04/ProjectINSIGHT.AI.git /opt/insight-ai/app
cd /opt/insight-ai/app
python3.13 -m venv venv
./venv/bin/pip install --upgrade pip
./venv/bin/pip install -r requirements-aws.txt
```

Create `/opt/insight-ai/.env` from `.env.example` and set `GROQ_API_KEY`.

```bash
sudo cp deploy/insight-api.service /etc/systemd/system/
sudo cp deploy/insight-ui.service /etc/systemd/system/
sudo cp deploy/nginx-insight-ai.conf /etc/nginx/sites-available/insight-ai
sudo ln -s /etc/nginx/sites-available/insight-ai /etc/nginx/sites-enabled/insight-ai
sudo nginx -t
sudo systemctl daemon-reload
sudo systemctl enable --now insight-api insight-ui nginx
```
